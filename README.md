# Cosette Parser

SQL parser for the [Cosette Solver](https://github.com/cosette-solver/cosette-rs)
based on that of [Apache Calcite](https://calcite.apache.org/).

## Requirements

Java version 11.

## Get Cosette Parser

### Download and build

```bash
$ git clone git://github.com/cosette-solver/cosette-parser.git
$ cd cosette-parser
$ ./mvnw install
```
